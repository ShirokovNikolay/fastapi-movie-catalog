# FastAPI Movie Catalog

## Develop

### Setup:

Right click `movie catalog` -> Mark Directory as -> Sources Root


### Install dependencies

Install all packages:
```shell
uv install
```

### Configure pre-commit

Install pre-commit hook:
```shell
pre-commit install
```

### Run

Go to workdir:
```shell
cd url-shortener
```

Run dev server:
```shell
fastapi dev
```
